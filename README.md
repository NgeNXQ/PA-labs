1. Клонувати репозиторій.
2. Перейти у директорію репозиторію і виконати команду dotnet run.


    ![image](https://user-images.githubusercontent.com/89969565/210489552-c048896e-34a7-4bc4-a2d0-73717e63dcfc.png)

3. Програма запрошує до введення деяких значень, які є необхідними для роботи алгоритму. У запрошенні до вводу вказані стандартні значення, які передбачені варіантом.
4. Програма виодить мінімальний шлях, який пройшла найкраща мураха на кожній ітерації. Також був виведений мінімальний шлях.
5. Завершення роботи з програмою.
