1. Клонувати репозиторій.
2. Перейти у директорію репозиторію і виконати команду dotnet run.


    ![image](https://user-images.githubusercontent.com/89969565/210111612-61e80fb6-6d8b-42ca-acf7-ed0f74596548.png)

3. Програма запрошує до введення назви лабіринту, можливі варіанти вводу:
maze1, maze2, maze3, maze4, maze5, maze6, maze7, maze8, maze9, maze10, maze11, maze12, maze13, maze14, maze15, maze16, maze17, maze18, maze19, maze20.

   ![image](https://user-images.githubusercontent.com/89969565/210111666-30a3ed71-62ea-4fad-a336-5151366483c1.png)

4. Далі програма просить ввести назву алгоритму пошуку. Варіанти вводу: RBFS, LDFS.

   4.1. Обрання алгоритму пошуку шляху RBFS.
   
      ![image](https://user-images.githubusercontent.com/89969565/210111684-ef2b58de-3b44-46e4-b63b-1a56fe85d713.png)
      
   4.2. Обрання алгоритму пошуку шляху LDFS.
   
      ![image](https://user-images.githubusercontent.com/89969565/210111697-bedb87e0-797a-4035-a58e-cf9322331055.png)
      
      4.2.1. У випадку обрання LDFS, програма запросить ввести заглиблення шляху пошуку. Це має бути додатнє ненульове значення.
      
      ![image](https://user-images.githubusercontent.com/89969565/210111718-942678cc-1b15-453c-a435-b9ae5a1dad14.png)

5. Завершення роботи з програмою.
