1. Клонувати репозиторій.
2. Перейти у директорію репозиторію "ExternalSorting".
3. Запустити за допомогою dotnet run.


    ![image](https://user-images.githubusercontent.com/89969565/210106626-6d0290b2-494a-453f-8bc9-aa63361e5c99.png)

4. Обрання алгоритму сортування. 0 - відповідає за звичайне зовнішнє, а 1 відповідає за попереднє сортування числових послідовностей.
    4.1. У випадку обрання попереднього сртування послідовностей, потрібно ввести кількість байт, які будуть попередньо відсортовані (число має бути більше 0)
5. Програма виконує сортування файлу за допогою алгоритму зовнішнього сортування.
6. Повідомлення про завершення сортування файлу. Вміст файлу можна переглянути за таким шляхом: PA-labs-feature-Labwork_1\ExternalSorting\A.txt.
