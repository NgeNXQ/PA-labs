1. Клонувати репозиторій.
2. Перейти у директорію проєкту.
3. Забілдити рішення за допомогою msbuild


    ![image](https://user-images.githubusercontent.com/89969565/210712581-11ea84df-1821-463a-b794-d5a4be2d1069.png)

5. .exe файл знаходиться у директорії "build" репозиторію.


    ![image](https://user-images.githubusercontent.com/89969565/210713018-cb3ba0b0-ae3a-45de-abe1-2a3c712846a4.png)

4. Перед виконанням "запитів" потрібно настиснути на кнопку "Load" щоб завантажити файли, у яких зберігаються записи.
    
    
    ![image](https://user-images.githubusercontent.com/89969565/210713150-63b9d890-0bf3-4607-8db5-df08b2a195db.png)
